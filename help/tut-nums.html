In your first MiniScript program, you printed the number 123 to the output.

You could print any number here, including ones with decimal points. Try this:

print 2.71828

But that's not all. MiniScript can also do math, using + for addition, - for subtraction, * for multiplication, and / for division. Try it! 

print 6 * 7

Note that MiniScript is not picky about spaces around such operators. Try changing "6 * 7" to "6*7" or even "6* 7" or any other variation. You should find they all produce the same result.

Ready for your first multi-line program? People often say 22/7 (22 divided by 7) is a decent approximation of pi. But how good is it? Let's find out, using the fact that pi is a built-in function in MiniScript.

print 22/7
print pi
print 22/7 - pi

You should have three lines of output. The first is the value of 22/7, approximated as a decimal. The second line is the value of pi (both to six decimal places). And the last line is the difference between them.

There are a few other operators and functions that work with numbers. But you don't need to worry about those for now. Let's move on!
