Text is stored in strings of Unicode characters. String literals are written within straight quotation marks. If you need to include a quotation mark in a string, write it twice. (No other escaping or special processing of certain characters is done.)

print "Hello, Jane.  I'm Bob."
print "Hello, ""Bob,"" if that is your real name."

Strings support the following operators:

+string concatenation
-string subtraction (chop)
*replication (i.e. repeat a string some number of times)
/division (get some fraction of a string)
==, !=, >, >=, <, <=comparison
[i]get character i (first character is 0)
[i:j]get substring ("slice") from i up to j

The indexes i and j above are nifty: if negative, they count from the end of the string. If nonnegative, they count forward, starting at zero. Also, with the slice operator, you can omit either or both values, and they default to the start and end of the string. Examples, all supposing that s = "abcde":

s[0]"a" (first character)
s[1]"b" (second character)
s[-1]"e" (last character)
s[-2]"d" (next-to-last character)
s[1:-1]"bcd" (everything from the second up to the last character)
s[1:]"bcde" (everything from the second character to the end)
s[:-1]"abcd" (everything up to the last character)

Slicing and indexing of strings (and lists) is incredibly powerful once you get the hang of it.

Note that strings, like numbers, are immutable: you can't actually change a string value. The operators above all return a new string (well, except for the comparison operators, which return numeric truth values). String comparisons in MiniScript are case-sensitive.
