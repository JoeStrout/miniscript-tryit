A for ... in statement loops over a block of code once for each value in a list. The list can be anything, but a common pattern is to use the range function to get a list of numbers. For example:

for i in range(10, 1)
   print i + "..."
   wait
end for
print "Liftoff!"

The whole set of lines from for to end for is called a for loop. On each iteration through the loop, the variable before in is assigned one value from the given list. Can you guess what the code above will do?

Instead of a list, it's also possible to iterate over a text string or a map. In the case of a string, the loop variable will be given each character of the string in order.

When using for on a map, it's a little more complicated. On each iteration, the loop variable is assigned a little mini-map containing key and value. 

m = {1:"uno", 2:"dos", 3:"tres"}
for kv in m
   print "Key " + kv.key + " has value " + kv.value
end for

This example prints out each of the key-value pairs in the map.
