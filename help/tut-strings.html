In addition to numbers, MiniScript can also work with letters and any other printable characters. In computer jargon, a group of characters is called a "string" because they are strung together in a fixed order. Try this:

print "Hello world!"

Run this program. You should see the words Hello world! appear in the output area. Again, if you get an error, check your spelling and punctuation very carefully and try again.

That "Hello world!"part of the code is a string. Strings in MiniScript must always begin and end with quotation marks — exactly the straight, double-type quotation marks shown; single-quotes or curly quotes won't work.

Between the quotation marks, you can type almost anything. Perhaps you're programming an old-school cartoon character who curses like this:

print "@*#&!"

But what if you want to print a quotation mark? There's a special trick for that. Within a string, you must type the quotation mark twice. MiniScript will see that and interpret it as just one quotation mark. Try it:

print "This --> "" <-- is a quotation mark!"

Challenge: Can you get MiniScript to print the following output? 

Hello, "Bob," if that is your real name.

Give it a try!
