MiniScript has four basic data types:

  • Numbers: values like 42, -5, and 3.14; also represent truth values.
  • Strings: chunks of Unicode text, like "Hello" and "日本語".
  • Lists: a sequence of other values, like [1, 2, 3].
  • Maps: a set of key-value pairs; also the basis of classes and objects.

(Technically, there is a fifth data type: functions. Functions are "first-class objects" in MiniScript and can be passed around like any other value. But most MiniScript users should not worry about this most of the time.)
