The if statement is how your code makes decisions in MiniScript. It simply defines a block of statements to execute when a given condition is true. It looks like this:

if 2+2 == 4 then
   print "math works!"
end if

This says that if two plus two is equal to four, then it should print "math works!"; otherwise, it should not (because in that case, math would be very broken!).

Notice that, unlike some programming languages, you do not need parentheses around the condition in an if statement (and using them is considered bad style).

You can have any number of statements inside the if block. And of course, if statements can be nested. When the condition is not true, MiniScript will skip down to the corresponding end if.
