Use a while loop to repeat a block of code as long as some condition is true.

s = "Spam"
while s.len < 50
   s = s + ", spam"
end while
print s + " and spam!"

This loop keeps adding to a string as long as the length is less than 50.

If you need to make a "forever" loop that never exits, use while true. (See the break topic for an example.)
