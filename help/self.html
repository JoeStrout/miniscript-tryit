When you invoke a function via dot syntax (e.g. x.foo), that function receives one extra argument, called self, that refers to the object (map) it was invoked on.

This allows classes to define methods that can access data on the instance. Consider the following.

Greeter = {"name": "anonymous"}
Greeter.greet = function()
    print("Hello, my name is " + self.name + "!")
end function

bob = new Greeter
bob.name = "Bob"
bob.greet

This prints "Hello, my name is Bob!" because self in this case is bound to the bob map. If we invoked the same function via some other map, it could have a different name.

mary = new Greeter
mary.name = "Mary"
mary.greet
