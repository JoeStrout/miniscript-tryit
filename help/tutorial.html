This tutorial provides a gentle, step-by-step introduction to programming with MiniScript. If you've never programmed before, or just want a bit of guidance as you learn your way around a new language, you've come to the right place.

You should see two big boxes above: the first one is a code editor, and the second one (with the black background) is the output area. (If you're on a desktop or laptop, and these don't appear side-by-side, try stretching the browser window out until they do.)

Click in the code editor, and you should get a blinking cursor. Select all the text in there, delete it, and then type the following.

print 123

Now click the Run button. You should see the number 123 appear in the output area.

If you got an error, don't worry! Just check your typing very carefully. Computers are picky about spelling, spacing, and capitalization. Everything must be exactly as shown.

Congratulations! You have written your first MiniScript program! It's not a very interesting program, but we'll fix that very soon.
