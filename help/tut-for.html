Now that you know about lists (and how to slice them), we can go over the second way of looping in MiniScript: the for loop. It looks like this:

for x in [2,4,8,16,32]
	print x
end for

The key bit is the first line above, which has the structure for someVariable in someList. The body of the loop — everything between for line and the end for line — gets repeated once for each element of the list. And on each repeat, the variable gets assigned the value of that element.

So this lets you easily do something for every element of a list.

A very common pattern is to use this with the range function, which builds a list of numbers from the arguments you give it. So if you just want to do something ten times, you can write:

for i in range(1, 10)
    print i
end for

The range function is really handy, especially when used with for. It can take a third parameter, which is the step size. What do you think range(0, 50, 5) would do? Experiment with different arguments until you're comfortable with it.
