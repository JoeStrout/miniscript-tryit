An if statement can be extended with an else clause to specify what should be done when the given condition is not true. Here's an example: 

age = val(input("Enter your age:"))
if age > 23 then
   print "Wow, you're old!"
else
   print "Hello, youngster!"
end if

This program asks the user to enter their age. If they enter a number greater than 23, it prints "Wow, you're old!" Otherwise — if they do not enter a number greater than 23 — then it prints "Hello, youngster!" It can never print both; it will always print either one or the other.
