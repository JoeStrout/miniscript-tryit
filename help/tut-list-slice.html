You've seen how to define lists, and get or set a single element thereof. You can also get a whole subset of a list at once, using the : slice operator. Basically, instead of specifying a single index to get, you specify a start and end index separated by a colon. This returns a new list containing all the elements from start up to (but not including) the end specified. For example:

x = [2, 4, 6, 8, 10, 12, 14]
print x[1:5]

This returns the portion of the list from element 1 (which is 4) up to but not including element 5 (the 12)... i.e., [4, 6, 8, 10].

You can use negative indexes to count from the end, too. See if you can you guess what this will print:

x = [2, 4, 6, 8, 10, 12, 14]
print x[2:-1]

Finally, either index may be omitted. If you leave out the first index, the result begins at the start of the list. If you leave out the last index, the result continues to the end of the list. So, for example, you could get the first three elements of a list like so: 

x = [2, 4, 6, 8, 10, 12, 14]
print x[:3]

Challenge: How would you get the last three elements of a list?

P.S. Slicing can also be done on strings! It works exactly the same way. Try it!
