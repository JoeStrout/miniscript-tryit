The continue statement skips the rest of the current for or while loop, continuing with the next iteration. It's often used near the top of the loop to skip items that you don't want to process.

s = "I never could get the hang of Thursdays."
result = ""
for c in s
   if c == "a" or c == "e" then continue
   result = result + c
end for
print result

This loop iterates over all the characters of s, adding them to result... but skips any character equal to "a" or "e".
