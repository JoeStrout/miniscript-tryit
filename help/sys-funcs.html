The following functions relate to the operation of MiniScript itself, or its host environment. Note that except for globals and locals, these functions are only quasi-standard, and may vary a bit depending on the host environment.

globalsreference to the global variable map
input(s="")display prompt s, wait for the user to enter a string, and return it
localsreference to the local variable map (of the current call frame)
print(x)convert x to a string and print to the default text output stream
timenumber of seconds since program execution began
wait(x=1)pause x seconds before proceeding
yieldwait for the next iteration of the main engine loop (e.g. next frame in a game)
